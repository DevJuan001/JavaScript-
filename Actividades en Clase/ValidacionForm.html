Atributos de Validacion del Formulario Html

1. required

Este sirve para especificar que un campo debe completarse antes de que se pueda enviar el formulario


2. minlength & maxlength

Este sirve para especificar la longitud minima y maima de los datos de texto


3. min y max

Este sirve para especificar los valores minimo y maximo de los tipos de entrada numerico


4. pattern

Este sirve para definir un patron que deben seguir los datos introducidos
