Metodos de Arrays

Metodos Transformadores

1. Metodo "push"

Este metodo se utiliza para agregar uno o mas elementos al final de un array

2. Metodo "pop"

Este metodo elimina el ultimo elemento de un Array y lo muestra

3. Metodo "shift"

Este metodo elimina el primer elemento de un Array y lo muestra

4. Metodo "Unshift"

Este metodo agrega uno a mas elementos al inicio del Array y lo muestra

5. Metodo "reverse"

Este metodo sirve para invertir el orden de los elementos de un Array

6. Metodo "sort"

Este metodo sirve para ordenar los elementos de un Array y lo muestra

7. Metodo "splice"

Este metodo sirve para agregar o eliminar elementos de un Array en una posicion especifica y lo muestra

Metodos Asesores

1. Metodo "join"

Este metodo une todos los elementos de una matriz o objeto en una cadena y lo muestra

2. Metodo "slice"

Este metodo devuelve una parte del Array de dentro un nuevo Array empezando por inicio hasta el fin

Metodos de Repeticion

1. Metodo "filter"

Este metodo crea un nuevo array con todos los elementos que cumplan la condicion

2. Metodo forEach

Este metodo ejecuta una funcion indicada una vez por cada elemento del Array
